The World Wide Web

The World Wide Web is a collection of web resource and application which can be accessed through the Internet.

Types of Web Resources

  • Static
  • Dynamic

Types of Web Applications

  • Web Server
    • Hosts the resources and handle the client requests.
  • Web Client
    • The ones accessing or utilizing the resources. Done on browsers or applications.

History of the World Wide Web

Who is Tim Berners-Lee?

Tim Berners-Lee is the father of the modern World Wide Web. The World Wide Web started a tool that Tim Berners-Lee wanted to help scientists easily share their data. At that point, hypertext and the internet already existed. Tim Berners-Lee just thought of another way to utilize it.

The World Wide Web Development Timeline

1989

The World Wide proposal was created by Tim Berners-Lee. His proposal included the three WWW main technologies:

  • HTML
  • URL
  • HTTP

1991

Tim Berners-Lee created the first website which was hosted by CERN in Switzerland where he used to work.

1992

The first web server was set up in America and there were only 10 known web servers at this point in time.

1993

The world wide web finally became open source. CERN allowed anyone to use the web protocols and code for free. People were able to create their own websites for their businesses. The known web servers at this point had a count of 50.

1994

The World Wide Web became accessible at people's own homes. Telecommunications providers gave that access. The World Wide Web Consortium (W3C) was also founded by Tim Berners-Lee which is an organization that makes sure that the World Wide Web keeps on improving.

WWW vs the Internet

People often get confused and think the World Wide Web and the Internet are the same thing. Well, they're not. The Internet is merely the infrastructure or network being utilized by the World Wide Web.

The Internet

Components

  • Hosts
    • The end devices or the devices utilized by both clients and servers.
  • Media
    • The wires or the medium that data travels or transmitted on.
  • Protocols
    • Protocols are standards for communication, connections and data transfer in a network.

How does the Web Work?

URL

  • Uniform Resource Locator
  • It is a unique identifier indicating the location of a resource in the web.

IP

  • Internet Protocol
  • Used by hosts and the Domain name server (DNS) would be the one giving out IP addresses to host names.
