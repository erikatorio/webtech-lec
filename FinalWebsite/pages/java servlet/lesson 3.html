Variable

A variable contains a data value and is assigned with a specific datatype. Here are the three types of variables in Java:

  • Local variable – it is declared inside the method
  • Instance variable – it is declared inside the class but outside the method and it is not declared as static.
  • Static variable – it is declared as static and it cannot be local.
