Express JS

Express JS is a Node.js web application framework that provides rich set of features for mobile and web applications. It is flexible, minimal and caters rapid development of Node based web applications.

Here are the main features of Express:

● Express allows to set up middlewares to respond to HTTP Requests.

● Express allows to dynamically render HTML Pages based on passing arguments to templates.

● Defines a routing table which is used to perform different actions based on HTTP Method and URL.
