Conditional structure

Sometimes we want to write our code with conditions wherein it will only execute if the condition have been met.

Here are some conditional statements:
If statement – if condition is true it will execute the code
If..else statement - if condition is true it will execute the code otherwise it will execute the code that lies on false.
If...elseif…else statement – if conditions is true it will execute the codes for multiple condtions
Switch statement – selects one of code to execute
