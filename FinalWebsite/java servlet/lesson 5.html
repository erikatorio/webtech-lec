HTTP Servlet

HttpServlet is an abstract class. It provides methods such as doGet, doPost, and many more. Here are some methods in HttpServlet:

  • doGet method handles the GET request.
  • doPost method handles the POST request.
  • doHead method handles the HEAD request.
  • doOptions method handles the OPTIONS request.
  • doPut method handles the PUT request.
  • doTrace method handles the TRACE request.
  • doDelete method handles the DELETE request.
  • getLastModified method returns the time when HttpServletRequest was last modified

To create a servlet, we will use Apache Tomcat Server. Follow this steps:

  1. Create a directory structure
  2. Create a Servlet

    3. The servlet can be created in any of these three ways:

    • By implementing Servlet interface,
    • By inheriting GenericServlet class
    • By inheriting HttpServlet class
  3. Compile the Servlet
  4. Create a deployment configuration file
  5. Start the server and project
  6. Access the servlet
