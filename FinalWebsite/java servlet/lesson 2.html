Java Servlet Life Cycle

Servlet Initialization

The init() method is only called once when the servlet is created

Servlet Request Handling

The service() method is called to handle requests coming from clients and responds back to clients.

The method calls the doGet method to handle GET requests and the doPost method to POST requests

Servlet Termination

The destroy() method is called when the life cycle of servlet ends

Java Servlet Variable Scope

Iteration level is for current server access only. This is the request and response level (Local Variables: doGet and doPost is used here)

User/Session level is used to store the user specific data

Application level is used to store global data which is going to be shared in the entire application
