NodeJS and HTTP Request

Running NodeJS

A NodeJS file is executed by issuing the node command followed by the NodeJS file name.

HTTP Module

NodeJS has a module named http which allows data to be transferred through HTTP or Hyper Text Transfer Protocol. A request is made by a client and a response is generated by the server.

Creating a Web Server

Servers can be created through the HTTP module by invoking the createServer and listen method.
