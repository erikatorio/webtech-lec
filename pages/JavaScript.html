JavaScript Execution Environment

Core JavaScript (EcmaScript) Language and API

(Traditional) Browser Object Model (BOM) API (non-standard)

  • window, navigator, screen, location, history

Document Object Model (DOM) API

  • Node, Document, Element, Text, Attr, Document Type, Comment, etc
  • Event, EventTarget, EventLister, etc
  • CSSStylesheet, CSS Rule, etc

Miscellaneous JavaScript Web APIs

  • Geolocation, IndexedDB, Local Storage, Push Notifications, Web Sockets, Web Workers ...

API that allows access to the HTML document from within scripts associated with the web page.

  • a parsed HTML document is represented by a DOM tree, which contains nodes representing elements, element attributes, textual content, and other HTML document components (e.g. DOCTYPE, comments, etc)

Accessed via the Document object properly of the global Window object

Types of JavaScript

Linked (external) script

<script type="text/javascript" src="javascript.js">

the defer and async keyword is used when you don't want interference with loading resources.

<script type="text/javascript" src="javascript.js" defer>
<script type="text/javascript" src="javascript.js" async>

Embedded (internal) scripts

Javascript inside an html document defined by the <script> tag

Inline Script

Incorporating script on a line of html code

<a href="#" onclick="alert('Welcome!')">

Functionalities

Retrieve references to nodes(or node collections) in the DOM

  • getElementByID()
  • getElementByTagName(), getElementByClassName(), getElementByName()
  • querySelector(), querySelectorAll()

Traverse the DOM tree (from a given node)

  • parentNode, parentElement
  • childNodes, children
  • firstChild, lastChild, nextSibling, previousSibling
  • firstElementChild, lastElementChild, nextElementSibling, previousElementSibling

construct/copy nodes

  • createElement(), createTextNode(), createAttribute(), etc.
  • cloneNode(), importNode()

manipulate the DOM tree

  • appendChild(), insertbefore(), replaceChild(), removeChild()
  • adoptNode()

miscellaneous attributes and methods

  • nodeType, nodeName, nodeValue
  • attributes,id, tagName, className, classList

JavaScript Language Fundamentals

Variables and Constants

var

Becomes a property of the window object.

var x = 1;

let

Does not become a propert of the window object.

let y = 2;

Primitive Data Types

  • Numbers
  • Strings
  • Boolean
  • null
  • undefined

Data types in JavaScript is dynamic. Meaning you don't have to specify the data type on your code.

Functions

Function syntax

function func1(parameters){
//code..
}

arguments object

Used when a function declaration has no parameters but you can dynamically add parameters when invoked.

function func2(a){
console.log(arguments[0]);
}

Arrays

Can be declared using the Array object, empty (array[]) or instantiation.

Dynamic size and allows "sparse" arrays.

0-based index

Allows array nesting

var heroes = [["Supergirl", "The Flash"], ["Iron Man", "Captain America"]]

Event Propagation Order

How an event propagates on your html document

  • capturing phase
  • at target
  • bubbling phase

var button = document.getElementByID('button1');
button.onclick(h);

You can also use the addEventListener() method.

button.addEventListener('click', h);
