HTML

HTML is short for Hypertext Markup Language. It used to created pages that can be viewed in the World Wide Web. Each page would then contain a series of connections between other pages with the use of hyperlinks. In the modern times, HTML is used for semantic specification of web page structure and content.

History

Pre-Standardization

  • HTML tags (1991) by Tim Berners-Lee
  • HTML Internet Draft by Tim Berners-Lee & Dan Connolly
  • HTML+ by Dave Ragett

The 1st Browser War

The first browser war occurred between Internet Explorer and Netscape. In 1995, Netscape Navigator was widely used and Microsoft gave license to Mosaic to create Internet Explorer. Once Internet Explorer was pre-installed with Windows, this gave them a wider audience.

HTML Standards

HTML 2.0

  • Date: Nov 1995
  • Standardized by: IETF

HTML 3.2

  • Date: Jan 1997
  • Standardized by: W3C

HTML 4.0

  • Date: Dec 1997
  • Standardized by: W3C

XHTML

XHTML is short for Extensible HyperText Markup Language and it was created to fixed what was then a very messy, malformed and non-standard HTML. It also provided interoperability between HTML and XML-based systems.

XHTML 1.0

  • Date: Jan 2000
  • Standardized by: W3C

XHTML 1.1

  • Date: May 2001
  • Standardized by: W3C

HTML5

When HTML5 was introduced in 2008, the fate of Flash hung in the balance. Then in November of 2011, Adobe announced that they would stop the production of Flash for mobile devices. While HTML5 kept growing and became widely accepted.
